Welcome!

This web based design tool has been developed within the Vinnova financed project Green Urban Systems project.

This tool is used for simultaneous design of applications and storm water solutions taking traffic loads and rain fall into account. The tool needs to be completed and expanded before it can be used for designing real applications. You are welcomed to register and evaluate this version.
